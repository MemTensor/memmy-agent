| 命令 | 说明 |
| --- | --- |
| `memmy` | 进入交互式聊天 |
| `memmy onboard` | 初始化配置和 workspace |
| `memmy onboard --wizard` | 终端菜单式配置 |
| `memmy status` | 查看配置、workspace、模型和 Provider 状态 |
| `memmy agent --message "..."` | 发送单轮消息 |
| `memmy serve` | 启动 OpenAI 兼容 API |
| `memmy provider login <provider>` | Provider OAuth 登录 |
