`/main` 是完整的 Agent 工作台：左侧任务列表，中间消息流，底部输入框。

## 输入与发送

- `Enter` 发送，`Shift + Enter` 换行；支持中文输入法组合态。
- `+` 按钮添加附件：最多 4 个，单文件 10MB，支持图片和 `.pdf`、`.docx`、`.xlsx`、`.pptx`、`.md`、`.csv`、`.json` 等常见格式。
- 麦克风按钮语音输入（需配置 ASR 模型），转写结果合并进输入框。
- 任务运行时发送按钮变为停止按钮。

## Slash command

输入 `/` 打开命令面板：

| 命令 | 作用 |
| --- | --- |
| `/new` | 开始新任务 |
| `/stop` | 停止当前任务 |
| `/restart` | 重新运行当前任务 |
| `/status` | 查看 Agent 运行状态 |
| `/model` | 查看当前模型配置 |
| `/goal` | 查看或管理目标状态 |
| `/dream` | 手动触发记忆整理 |
| `/dream-log` | 查看整理记录 |
| `/dream-restore` | 恢复整理前状态 |
| `/help` | 帮助 |

## 任务列表

按置顶、今天、昨天、最近 7 天、更早、已归档分组。支持重命名、置顶、归档、恢复、删除；运行中任务显示旋转状态，完成未读任务显示未读点。`Cmd/Ctrl + K` 呼出任务搜索。
