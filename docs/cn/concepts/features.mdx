## 亮点

六种外部 Agent 来源，扫描去重入库多会话、附件、语音、Slash command透明置顶小窗，随叫随到20+ 托管集成 + 自定义 MCP6 种 IM 渠道接入15+ Provider，自有额度。

## 完整列表

### 记忆

- 外部来源扫描：Cursor、Claude Code、Codex、Opencode、OpenClaw、Hermes
- 记忆面板：概览、条目、任务、策略、世界模型、技能、分析、日志
- Dream 整理：手动 / 周期触发，可查看日志和回滚
- memmy-memory CLI 与 Memory Runtime HTTP API
- 外部 Agent Skill 安装（主动读写记忆）

### 桌面端

- 多会话任务列表：分组、置顶、归档、搜索、未读提示
- 流式输出、停止 / 重启、Agent 状态面板
- 附件（图片 + 常见办公文档）、语音输入（ASR）
- Slash command、Agent 产物预览
- 桌宠模式：输入气泡、多任务气泡、回答预览、会话续接

### 模型与认证

- 账号模式（平台 Token）与 API Key 模式（BYOK）
- 主模型、Embedding、记忆摘要、技能进化、ASR、图像生成六类模型独立配置
- 15+ Provider，含 OAuth 登录

### 工具与自动化

- 内置工具：文件、shell、Web 搜索与抓取、定时任务、长任务、图像生成等
- Composio 托管集成 + 自定义 MCP server（stdio / sse / streamableHttp）
- 6 种消息渠道

### 接口

- OpenAI 兼容 API（`memmy serve`）
- Memory Runtime API、桌面本地 API、SSE 事件流
