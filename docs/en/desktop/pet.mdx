## Interactions

- Single-click to expand the input bubble; `Enter` to send; voice recording with transcription is supported.
- Double-click to leave pet mode and open the full workbench.
- Drag with the left mouse button to move the window; the right-click menu can open full mode.
- When tasks are running or there are unread completions, a numeric badge appears; click it to expand a mini task list (up to 6).

## Session Continuation

- If a task is running, new messages continue the same session; tasks completed within the last 5 minutes also continue their session.
- After clicking "New session", the next message creates a new session.
- The pet and full mode share the WebUI thread, so context survives switching.
