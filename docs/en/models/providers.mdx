title: Supported Providers
icon: Cpu
---

The Agent Runtime currently supports:

OpenAI-compatible APIs, Anthropic, Azure OpenAI, AWS Bedrock, OpenRouter, DeepSeek, Gemini, Zhipu, DashScope, Kimi, MiniMax, Ollama, LM Studio, GitHub Copilot, OpenAI Codex OAuth, and more.

OAuth-based providers can sign in via the CLI:

```bash
memmy provider login <provider>
```
