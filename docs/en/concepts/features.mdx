## Highlights

- Six external Agent sources — scanned, deduped, ingested
- Multi-session, attachments, voice, slash commands
- A transparent always-on-top mini window, on call anytime
- 20+ managed integrations + custom MCP
- 6 IM channels
- 15+ providers with your own quota

## Full List

### Memory

- External source scanning: Cursor, Claude Code, Codex, Opencode, OpenClaw, Hermes
- Memory panel: overview, entries, tasks, policies, world model, skills, analytics, logs
- Dream consolidation: manual / scheduled, with logs and rollback
- memmy-memory CLI and the Memory Runtime HTTP API
- External Agent Skill installation (active memory read/write)

### Desktop

- Multi-session task list: grouping, pinning, archiving, search, unread indicators
- Streaming output, stop / restart, Agent status panel
- Attachments (images + common office documents), voice input (ASR)
- Slash commands, Agent artifact preview
- Pet mode: input bubble, multi-task badge, answer preview, session continuation

### Models & Auth

- Account mode (platform tokens) and API Key mode (BYOK)
- Six independently configurable model slots: primary, Embedding, memory summary, skill evolution, ASR, image generation
- 15+ providers, including OAuth sign-in

### Tools & Automation

- Built-in tools: files, shell, web search and fetching, scheduled tasks, long tasks, image generation, and more
- Composio managed integrations + custom MCP servers (stdio / sse / streamableHttp)
- 6 messaging channels

### APIs

- OpenAI-compatible API (`memmy serve`)
- Memory Runtime API, desktop local API, SSE event stream
