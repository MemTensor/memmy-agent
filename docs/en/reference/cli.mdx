---

| Command | Description |
| --- | --- |
| `memmy` | Start an interactive chat |
| `memmy onboard` | Initialize config and workspace |
| `memmy onboard --wizard` | Terminal menu-driven setup |
| `memmy status` | Check config, workspace, model, and provider status |
| `memmy agent --message "..."` | Send a single-turn message |
| `memmy serve` | Start the OpenAI-compatible API |
| `memmy provider login <provider>` | Provider OAuth sign-in |
