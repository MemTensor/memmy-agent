`memmy serve` listens on `http://127.0.0.1:18990` by default:

| Method | Path |
| --- | --- |
| `GET` | `/health` |
| `GET` | `/v1/models` |
| `POST` | `/v1/chat/completions` |

Any tool or script that speaks the OpenAI Chat Completions protocol can reuse Memmy directly.
