| Service | Default Port |
| --- | --- |
| Memory HTTP / memory foundation | `18960` |
| Gateway health | `18970` |
| WebUI / WebSocket / admin HTTP | `18980` |
| OpenAI-compatible API (`memmy serve`) | `18990` |
| Desktop frontend Vite dev server | `19000` |
| Vite HMR | `19010` |
| Desktop local API | Random port |
| Composio MCP bridge | Reuses the desktop local API port |
